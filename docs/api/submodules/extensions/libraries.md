# Libraries

Tools used in programs to interact with web content or automate browsing tasks.

- `Apache-HttpClient`
- `Axios`
- `go-http-client`
- `got`
- `GuzzleHttp`
- `Java`
- `Java-http-client`
- `jsdom`
- `libwww-perl`
- `lua-resty-http`
- `Needle`
- `node-fetch`
- `node-superagent`
- `okhttp`
- `PHP-SOAP`
- `PostmanRuntime`
- `python-requests`
- `python-urllib`
- `Scrapy`
- ... etc.

## Code Example

```js
import { Libraries } from 'ua-parser-js/extensions';

const libParser = new UAParser(Libraries);
```
