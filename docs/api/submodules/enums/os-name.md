## Code Example

```js [greet-amigaos.js]
import { UAParser } from 'ua-parser-js'; 
import { OSName } from 'ua-parser-js/enums';

if (UAParser().os.is(OSName.AMIGA_OS)) {
    console.log('Hola, AmigaOS!');
}
```
